# 📖 AI RSS Reader 導入・利用ガイド

このアプリは、お気に入りのウェブサイトの更新情報を一箇所でまとめてチェックし、  
**Google の最新 AI「Gemini」** を使って記事内容を瞬時に要約できる **RSSリーダー** です。

---

## 🛠 基本的な使いかた

### フィードを登録する
アプリ内の入力欄に、取得したいサイトの **RSS URL** を入力し、登録ボタンを押してください。

### 記事を読み込む
登録が完了すると、自動的に最新の記事がリストアップされます。  
手動で更新したい場合は **「更新」ボタン** をクリックしてください。

### 記事を読む
リストから気になる記事を選択すると、詳細が表示されます。  
元のサイトで全文を読みたい場合は **「サイトで見る」リンク** をクリックしてください。

---

## 🔑 1. Gemini APIキーの取得方法

AI要約機能を利用するには、**Google AI Studio** から無料の API キーを取得する必要があります。

1. **Google AI Studio** にアクセスします  
2. Google アカウントでログインします  
3. 左側サイドバーの **「Get API key」** をクリックします  
4. **「Create API key in new project」** をクリックします  
5. 発行された API キーをコピーします  

⚠️ **この API キーは外部に公開しないようご注意ください。**

---

## 🛠️ 2. アプリへの設定方法

取得した API キーをアプリに登録します。

1. アプリ画面右上の **歯車アイコン（⚙️）** をクリックし、設定エリアを開きます  
2. **「AI Studio API Key」** の入力欄に、コピーしたキーを貼り付けます  
3. **「保存」ボタン** をクリックします  

API キーは **ブラウザ内（LocalStorage）** に保存されるため、  
ページを閉じても再入力する必要はありません。

---

## ✨ 3. AI要約機能の使い方

### 記事の読み込み
画面中央に表示されている記事一覧から、気になるニュースを選択します。

### 要約の実行
各記事カードの右下にある **「✨ 要約」ボタン** をクリックします。

### 要約の確認
モーダルウィンドウが開き、AIが記事の内容を  
**最大3項目の箇条書き** で分かりやすく表示します。

---

## 📝 その他の機能

- **フィードの追加**  
  左サイドバーの入力欄に RSS URL を入力し、「追加」をクリックすると  
  自分専用のニュースリストを作成できます。

- **フィードの削除**  
  登録した URL の横にある **ゴミ箱アイコン** をクリックすると削除できます。

- **情報の更新**  
  画面上部の **「更新」ボタン**、またはページの再読み込みで  
  最新の記事を取得します。

## ⚠️ 注意事項・制限

### APIの制限
無料枠の API キーを使用している場合、短時間に連続して  
**「要約」ボタンを押しすぎると、Rate Limit エラー** が発生することがあります。  
その場合は、少し時間を置いてから再度お試しください。

### プライバシー
入力した API キーは **あなたのブラウザ内にのみ保存** され、  
開発者を含む第三者に送信されることはありません。
