# 📖 アプリご利用ガイド

このアプリは、お気に入りのウェブサイトの更新情報を一箇所でまとめてチェックできる **RSSリーダー** です。

---

## 🛠 基本的な使いかた

### フィードを登録する
アプリ内の入力欄に、取得したいサイトの **「RSS URL」** を入力し、登録ボタンを押してください。

### 記事を読み込む
登録が完了すると、自動的に最新の記事がリストアップされます。  
手動で更新したい場合は **「更新」ボタン** をクリックしてください。

### 記事を読む
リストから気になる記事を選択すると、詳細が表示されます。  
元のサイトで全文を読みたい場合は **「サイトで見る」リンク** をクリックしてください。

---

## 💾 データの保存について（重要）

本アプリは、登録したフィードの情報を **お使いのブラウザ（ローカルストレージ）** に保存しています。  
以下の点にご注意ください。

### 端末ごとの管理
データは「今操作している端末」の「今使っているブラウザ」にのみ保存されます。  
スマホで登録したリストをパソコンで見ることはできません。

### ブラウザの変更
同じパソコンでも、Chromeで登録したデータはSafariやEdgeには引き継がれません。

### キャッシュの削除
ブラウザの履歴やキャッシュを完全に消去すると、登録したリストも削除される場合があります。

## ⚠️ ご注意：取得できないサイトについて

本アプリは、ウェブサイトが公式に配信している **「RSSフィード」** を利用して情報を取得します。

### RSSを公開していないサイト
サイト運営者がRSSを配信していない場合、そのサイトの情報を本アプリで取得することはできません。

### RSSの探し方
サイト内に **オレンジ色のアイコン** や **「RSS」** というテキストリンクがあるか確認してください。

### 技術的な制限
一部のサイトはセキュリティ設定（CORS制限など）により、  
直接アプリから読み込めない場合があります。あらかじめご了承ください。
